1-Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

Las tablas HTML se crean usando las etiquetas table.

¿Qué etiqueta se utiliza para definir una fila en una tabla?

Table Row (tr). Etiqueta contenedora de cada fila de la tabla.

¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

Table Header (th). Cada una de las celdas de cabecera de la tabla.

¿Cuál es la etiqueta utilizada para crear una celda de datos en una tabla?

Se utiliza td, para crear celdas de datos.

¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

colspan="x" extiende una celda dos columnas más allá.

¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

Rowspan. Será el atributo son los intervalos de fila, hacen que la celda se extienda hacia abajo en varias filas

Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

Se usa la etiqueta tr 10 veces, cada una de estas representará una fila dentro de la tabla.
